### DJV Bootcamp de FOPEA
Esta investigación periodística basada en datos fue elaborada durante el segundo semestre de 2025 en el ámbito de Data Journalism Visualization Bootcamp del Foro de Periodismo Argentino. El programa recibió el apoyo de la Embajada de Suiza en la Argentina.

### Análisis de datos
Para la limpieza y el procesamiento de datos se utilizó Python v. 3.12.12. De manera complementaria se usó Microsoft Excel v. 2021 para tareas de baja complejidad.

### Estructura y diseño
La estructura se definió en HTML, organizando el contenido en bloques secuenciales de texto, imágenes y visualizaciones embebidas (Flourish), lo que permite que la historia se despliegue progresivamente a medida que el usuario avanza con el scroll. El diseño y la jerarquía visual se resolvieron mediante CSS, controlando tipografías, colores, espaciados y comportamientos responsive. La distribución del contenido se optimizó para desktop y mobile. La interactividad y el contenido dinámico se implementaron en JavaScript, a partir de la carga de archivos JSON con datos previamente procesados. Las visualizaciones de datos de mayor complejidad se integraron mediante embeds de Flourish, permitiendo combinar desarrollo a medida con gráficos interactivos externos.
